# SunnyBill Logo Assets

Dieses Verzeichnis enthält alle Logo-Assets für die SunnyBill-Anwendung.

## Logo-Dateien

### Haupt-Logo
- **sunnybill-logo.svg** (600x180px) - Standard-Logo für die Filament-Sidebar und allgemeine Verwendung
- **sunnybill-logo-large.svg** (900x240px) - Größere Version für Startseiten und Header

### Favicon
- **favicon.svg** (32x32px) - Vereinfachtes Logo als Favicon für Browser-Tabs

## Logo-Design

Das SunnyBill-Logo kombiniert:
- **Sonne**: Symbolisiert Solar-Energie mit strahlenden Linien und einem freundlichen Gesicht
- **Rechnung/Dokument**: Repräsentiert die Rechnungsstellung und Dokumentenverwaltung
- **Farbschema**: 
  - Goldene Farbverläufe (#FFD700 → #FFA500 → #FF8C00) für die Sonne
  - Dunkle Grautöne (#2C3E50 → #34495E) für den Text
  - Blaue Akzente (#3498DB) für das Dokument

## Verwendung in der Anwendung

Das Logo wird automatisch in folgenden Bereichen verwendet:
- Filament Admin Panel Sidebar
- Login-Seite
- Browser-Favicon
- Alle Filament-Komponenten über die Logo-Komponente

## Technische Details

- Format: SVG (Scalable Vector Graphics)
- Skalierbar ohne Qualitätsverlust
- Optimiert für Web-Verwendung
- Unterstützt moderne Browser
